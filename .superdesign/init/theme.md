# Theme

## CSS Variables (`src/style.css`)

```css
:root {
  --font: "Instrument Serif", serif;
  --white: #e8e4df;
  --white-dim: rgba(232, 228, 223, 0.3);
  --white-faint: rgba(232, 228, 223, 0.12);
  --black: #000;
  --ease: cubic-bezier(0.4, 0, 0.2, 1);
  --ease-out: cubic-bezier(0, 0, 0.2, 1);
}
```

## Font

- Family: Instrument Serif (Google Fonts)
- Weights: Regular, Italic
- Usage: All text — input, hints, errors

## Colors

- Background: `#000` (pure black)
- Text: `#e8e4df` (warm off-white)
- Dim text: `rgba(232, 228, 223, 0.3)`
- Faint: `rgba(232, 228, 223, 0.12)`

## No Tailwind — plain CSS only.
